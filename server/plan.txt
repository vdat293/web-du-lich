## 📝 Ghi Chú

1. **Coupon**: Hiện tại Payment.jsx hardcoded `discountCodes` object - cần thay bằng gọi API `/api/coupons?code=XYZ`

2. **Wishlist**: Nút "Lưu" trong Details.jsx (dòng 132-136) chỉ là UI, chưa gọi API

3. **Payment**: Sau khi tạo booking thành công, cần gọi API `/api/user/payments` để xử lý thanh toán

4. **Reviews**: Phần reviews trong Details.jsx hiện là static (mock data), cần lấy từ DB

5. **Authentication**: Đã hoàn chỉnh, sử dụng JWT trong localStorage
